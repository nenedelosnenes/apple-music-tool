This project will with the users permission download their playlists or library from apple music locally. This project can be extended with the use of a raspberry pi or arduino to make your own mp3 player.

Phase 1:
  - Downloads the mp3 file from youtube using search parameters 
  - Also started work on the apple music intergration
    - Needs to be able to extract all of the names of songs and their author into a text file that the code can read and download each file one by one
Phase 2 and beyond:
  - Be able to Download the enire library 
  - Make search better (right now its so so)
  - Be able to download a selected playlist 
  - Include a better CLI experience
